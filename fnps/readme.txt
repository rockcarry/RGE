fnps - pixel & scan line functions

in this dir we put these files:

ftps.c   - solid pixel functions for all color format
ftpa.c   - alpha pixel functions for all color format
ftss.c   - solid scan line fill functions for all color format
ftsa.c   - alpha scan line fill functions for all color format
ftsbfc.c - bmp scan line fill functions for all color format without color convert
ftsbcc.c - bmp scan line fill functions for all color format with color convert
ftsbma.c - bmp scan line fill functions for all color format with mask color & alpha blending
pfspmp.c - scanline functions for pal map color & pal map bmp output
